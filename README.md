# gt4lsc
Renderer and editor for Live Sequence Charts in GToolkit.

## Installation

```st
Metacello new
	repository: 'github://isegorg/gt4lsc:main/src';
	baseline: 'Gt4Lsc';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfGt4Lsc asClass loadLepiter
```
